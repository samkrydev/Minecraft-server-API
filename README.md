# Minecraft Server API

A simple REST API built with Node.js and Express for managing Minecraft servers and players.  
Data is stored in an in-memory database, making the API lightweight and easy to run.

## Features
- Create, update and delete Minecraft servers
- Add players to a server
- Retrieve and modify player data
- Minimal, fast and beginner-friendly API structure

## Installation
npm install

## Development

npm run dev

## Production

npm start

The API runs on `http://localhost:3000`.
